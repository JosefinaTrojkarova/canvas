# Pixel Canvas

A real-time collaborative pixel art canvas application.

## Setup Instructions

### Server

Navigate to the server directory, install dependencies, and start the server:

```bash
cd server
npm install
node index.js
```

The server will run on port 3000 (or the port defined in your configuration).

### Client

Navigate to the client directory, install dependencies, and start the development server:

```bash
cd client
npm install
npm run dev
```

The client application will typically be available at `http://localhost:5173`.
